# ADR-0001 — Public Repository Scope

## Status

Accepted

## Context

This repository is public and must demonstrate engineering maturity without exposing private projects, customer information, internal architecture, confidential business rules or security-sensitive details.

## Decision

The repository will contain generic, anonymized and educational architecture material. Any code or diagram must remain safe to publish.

## Consequences

- The repository can be used as a professional portfolio artifact.
- Some implementation details will be intentionally simplified.
- Documentation quality becomes the main proof of maturity.
